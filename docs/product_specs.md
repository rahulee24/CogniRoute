# Product Specifications and Security Standards

## Compliance and Auditing
- **SOC 2 Type II**: The platform undergoes annual SOC 2 Type II audit examinations for security, availability, and confidentiality.
- **ISO 27001**: Fully ISO/IEC 27001:2022 certified.

## Data Encryption
- **In Transit**: All connections utilize TLS 1.3 encryption protocol.
- **At Rest**: Customer database files and indexing data are encrypted using AES-256 standards.

## Identity and Access
- **SSO**: Enterprise plans include integration for SAML 2.0 Single Sign-On (Okta, Azure AD, Ping Identity).
- **Audit Logging**: Full audit trail logs are available to admins on Enterprise tiers.
